

# Assignment 3
# Multi-Task Vision System using YOLO

## Objective
The objective of this project is to implement a multi-task computer vision system using YOLOv8 for:
- Object Detection
- Image Classification
- Pose Estimation
- Oriented Bounding Boxes (OBB)

## Tools Used
- Python
- Ultralytics YOLOv8
- VS Code
- Roboflow

## Tasks Performed

### 1. Object Detection
A YOLOv8 nano model was trained on a Roboflow dataset to detect objects using bounding boxes.

### 2. Image Classification
A classification model was trained to classify images into categories such as cats and dogs.

### 3. Pose Estimation
A pose model was trained to detect human keypoints and generate skeleton structures.

### 4. Oriented Bounding Boxes (OBB)
Attempted using Roboflow dataset. Dataset format limitations were observed, and standard detection was demonstrated.

## Deployment
All models were tested locally using command-line inference and webcam.

## Conclusion
The project successfully demonstrates multiple computer vision tasks using YOLOv8 on a local system without cloud resources.
